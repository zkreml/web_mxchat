Aktualizace Mxchat

  • Synapse: v1.135.0 → v1.136.0

Matrix Rooms v12

11. srpna bylo vydáno koordinované bezpečnostní oznámení a 14. srpna následovalo zveřejnění detailů.

Co je potřeba vědět

  • V ohrožení jsou pouze veřejné místnosti.
  • Opravy jsou dostupné jen pro místnosti v12.
  • Jakmile je server aktualizován alespoň na Synapse v1.135.1, bude používat v12 jako výchozí pro všechny nově vytvořené místnosti.

Podpora v klientech:

  • Synapse v1.135.1+ → obsahuje opravu
  • Element Web v1.11.109+ → podporuje v12
  • Cinny Web v4.9.0+ → podporuje v12
  • ShildiChat Web → zatím nepodporuje v12
  • FluffyChat Web → zatím nepodporuje v12
  • Mobilní aplikace → zkontrolujte poznámky k vydání. Pokud byla naposledy aktualizována v červnu nebo není zmíněna podpora v12, aplikace nepodporuje v12.

Co můžete udělat

  • Pokud máte veřejné místnosti, prostudujte si nový návod Matrix Foundation a naplánujte upgrade.
⚠️ Upgrade místnosti je rušivý proces – ovlivní uživatele a způsobí problémy.
Nedělejte zatím upgrade, pokud to není nutné – ne všechny servery jsou aktualizovány a může se stát, že se do místnosti všichni uživatelé nepřipojí.
